# Tasks 

# Création d'une page web pour les produits : (Done)
    La page doit permettre aux utilisateurs de choisir un produit, de sélectionner la quantité souhaitée et de remplir les informations de l'acheteur, 
        telles que le nom, l'adresse et le numéro de téléphone.
    Les fonctionnalités attendues sont basées sur le design partagé via Figma, qui détaille toutes les fonctionnalités à implémenter.
    Le développement de la page doit être réalisé en utilisant React.js pour le frontend.

# Création d'un panel admin :
    Le panel admin doit permettre à l'administrateur de consulter toutes les commandes passées sur le site et de modifier les informations relatives à ces commandes.
    Les fonctionnalités attendues sont également fournies dans le design partagé via Figma.
    Le développement du panel admin doit être réalisé en utilisant React.js pour le backend.
